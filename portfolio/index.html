Parte de mi formación como desarrollador fue planificar y ejecutar la realizacion de un sistema a elección como parte de mi educación en el mundo de las nuevas tecnologías.

Para cumplir con este proposito, se creo BrewMaster, un software en formato desktop, atravez de Java SE, utilizando NetBeans 11 como IDE. El mismo viene a cubrir la necesidad de Cerveza Artesanal Timoteo a la hora de llevar cuenta de los lotes, clientes, tipos de envases y estilos, asi como tambien realizar remitos y un elaborado reporte de ventas o producción mensual.

El mismo se encuentra todavia en desarrollo.

BrewMaster fue realizado por una sola persona, sin la utilización de frameworks, para de esta forma poder conocer bien a fondo cada libreria y aprender de ello. El motor de base de datos utilizado es MySQL
